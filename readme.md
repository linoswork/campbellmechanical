# Campbell Mechanical
A modern Drupal 10 site built from the ground up using DDEV (PHP 8.2).
This project focuses on performance, maintainability, and a clean content-editor experience.

## Highlights
- Custom block-based architecture (no Paragraphs)
- Tailwind CSS integration for design consistency
- Config-sync directory outside web root
- SEO + Open Graph optimization
- Built for deployment on shared hosting (A2 Hosting)
- Version-controlled via Git for clean config management

## Stack
- Drupal 10, PHP 8.2
- DDEV local environment
- Composer dependency management
- Git-based CI/CD workflow
- Tailwind CSS for front-end styling
